  • Blockchain creation project with Python

    • It is about the creation of a Blockchain in Python which we are going to develop an application that allows users to share information through a blockchain. The steps would be: Storing the transactions in blocks, Adding the fingerprints, Chaining the blocks, Implementing a proof-of-work algorithm, Adding blocks to the chain, Creating the interfaces, Establishing consensus and decentralization We would like to prevent any kind of tampering with the information stored within a block. To detect if the information in a block has been tampered with, we can use cryptographic hash functions.
